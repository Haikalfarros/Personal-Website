About Me

Ketertarikan saya pada pembuatan website dimulai ketika saya pertama kali melihat bagaimana sebuah halaman web dapat memberikan informasi dan interaksi yang menarik kepada pengunjung. Hal ini memotivasi saya untuk memahami lebih dalam tentang bagaimana website dibuat, mulai dari tampilan depan (front-end) hingga bagian belakang (back-end).

Dalam perjalanan belajar saya, saya telah mengeksplorasi beberapa teknologi dan alat yang digunakan dalam pembuatan website, antara lain:

HTML

(HyperText Markup Language): Bahasa dasar untuk membuat struktur sebuah halaman web. Saya telah mempelajari cara menulis tag-tag HTML dan bagaimana mengatur elemen-elemen pada halaman web.

CSS

(Cascading Style Sheets): Bahasa yang digunakan untuk mendesain dan memperindah tampilan halaman web. Saya belajar bagaimana menggunakan CSS untuk memberikan gaya, tata letak, dan responsivitas pada website.

JAVASCRIPT

Bahasa pemrograman yang digunakan untuk menambah interaktivitas pada halaman web. Saya telah mempelajari dasar-dasar JavaScript dan bagaimana menggunakannya untuk membuat website lebih dinamis.

Framework dan Library

Saya juga mulai mempelajari beberapa framework dan library populer seperti Bootstrap untuk mempermudah pengembangan front-end untuk memanipulasi DOM dengan lebih efisien.

Selain mempelajari teknologi-teknologi tersebut, saya juga aktif mengikuti tutorial online, membaca artikel untuk terus mengasah keterampilan saya. Saya percaya bahwa dengan dedikasi dan kerja keras, saya dapat menjadi seorang web developer yang handal di masa depan. Saya sangat antusias untuk terus belajar dan mengembangkan keterampilan saya dalam pembuatan website. Saya yakin bahwa pengalaman ini akan sangat berharga bagi masa depan saya, baik dalam karir maupun dalam kontribusi saya terhadap dunia digital.
